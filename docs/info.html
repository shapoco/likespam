対策方法

フォローしていないユーザーからの通知を切ることを検討してください。 大量の捨てアカウントを使って通知を送るこのスパムの性質上、 あなたの通知欄やいいね欄が汚されることを防ぐのは X 社による根本的な対策がとられない限り不可能に思えます。

深く考えるのはやめよう

関連キーワードの検索結果を見ているとこのスパムに対する怒りのポストが多く見られます。 いいね欄を汚されるのが不快なのは理解しますが、スパムに対して感情の炎を燃やすのは個人的には時間と精神力の無駄と思います。 「スパムからいいねが付いて不快だからポストしなおした」というのもよく見ますが、ブロック＋リロードして見えなくなれば十分だと思います。

スパムのプロフィールを見てすぐに「こんな幼稚な詐欺に引っかかる人がいるのか」と思ったあなたは、恐らく彼らの眼中にはありません。 彼らはそのような判断ができない (場合によっては社会的サポートを要するような) 人々をターゲットにしている可能性があります。

そういうわけで多くの人にとっては蚊やハエのような存在でしかないので、関わりたくない場合は通知を切るのがベストだと思います。 そうでなければ自分に来た通知を機械的に淡々とブロックまたは報告するしかありませんが、数が膨大なので無意味かもしれません。 何かしたくてたまらない方は捨てアカウントだけではなく以下の本体アカウントとその周辺アカウントを報告するか、 イーロンマスクに怒りをぶつけてください。

このサイトもそう長く続けるつもりはありませんので、スパム対策サイトであるかのように宣伝するのものご遠慮頂けると助かります。
