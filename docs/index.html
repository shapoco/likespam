いいねスパム一覧

概要

X において不特定多数に対して「いいね」通知を送りつけて詐欺 (?) に誘導しようとするスパムアカウントをまとめたものです。 管理者は勝手に「いいねスパム」と呼んでいます。以前はアカウント名に【公式】と付けていたことから、X では「公式スパム」などとも呼ばれています。

使用されているアイコン:

これらのスパムアカウントは数千個あり、もはやひとつひとつスパム報告することに意味があるのかは微妙です。

詐欺の窓口となっている @Zy_Iove は相当な数の報告を受けているはずですが X 社の怠慢のためか 凍結される気配がありません。 定期的に協力者に 本当に金品が貰えたかのようなリプライ を送らせているようです。

このページでは、X で観測されたスパムアカウントを収集しています。

いいねスパムの見つけ方

  • ハッシュタグ #いいねスパム を使った情報共有（ご協力お願いします）
  • キーワード検索でも見つかりますが、新しいアカウントが検索結果に反映されるのにはタイムラグがあり、 犯人はその前にアカウントと名前を切り替えてしまうため、現在活動中のアカウントを検索で網羅的に見つけるのは困難です。
    • X検索: "このアカウントが出てきたらおめでとう"
    • X検索: "書かれてたらすごい事が起こります"
    • X検索: "色々なギフト券が"
    • X検索: "特別なご案内です"
    • X検索: "個された人"

スパムからいいねが来たら…

  • スパムアカウントの ID を #いいねスパム タグでシェアしてください。情報共有にご協力お願いします。
  • 凍結されているのを見つけた場合もお知らせ頂けると助かります。
  • 匿名で報告したい場合は こちらの Google Form をご利用ください。

アカウント一覧 (参考)

これまでに観測されたスパムアカウントを 20 件ずつのグループにまとめて掲載しています。 PC 向けの 表形式版 もあります。

⚠ ブロックや X への報告が目的であれば、まずは #いいねスパム タグを使ってアカウントを探してください。 アカウントの数は膨大で、全てを一気にブロックしたり報告しようとするのは無謀です。 X では短時間に数十以上のアカウントをブロックすると閲覧が制限されたり強制ログアウトさせられたりします。 もし連続でブロックしたり報告したりする場合は、20～40 件毎に 1 時間程度空けてください。

スマホアプリの場合は各リンクをクリック後「ユーザー」のタブを選択してください。

灰色ハッチングは凍結確認済のアカウントです。
