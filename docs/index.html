アカウント一覧 (参考)

これまでに観測されたスパムアカウントを 20 件ずつのグループにまとめて掲載しています。 PC 向けの 表形式版 もあります。

⚠ ブロックや X への報告が目的であれば、まずは直近で活動しているアカウントを #いいねスパム タグ を使って探してください。 アカウントの総数は膨大で、全てを一気にブロックしたり報告しようとするのは無謀です。 X では短時間に数十以上のアカウントをブロックすると閲覧が制限されたり強制ログアウトさせられたりします。 もし連続でブロックしたり報告したりする場合は、20～40 件毎にしばらく時間を空けてください。

スマホアプリの場合は各リンクをクリック後「ユーザー」のタブを選択してください。

  • 打ち消し線: 凍結確認済 (水色 : 過去24時間以内に凍結確認)
  • 緑色 : リポストスパム (活動停止中。現状は無視していいと思います)
