いいねスパム一覧

概要

X において不特定多数に対して「いいね」通知を送りつけて 当選詐欺 (?) に誘導しようとするスパムアカウントをまとめたものです。 管理者は勝手に「いいねスパム」と呼んでいます。以前はアカウント名に【公式】と付けていたことから、X では「公式スパム」などとも呼ばれています。

使用されているアイコン:

このページでは X で観測されたスパムアカウントを収集しています。

これらのアカウントは1万個以上あり、もはやひとつひとつブロックすることに意味があるのかは微妙です。 報告するのであれば、詐欺の窓口となっている @Zy_Iove とその周辺のアカウント を報告することをおすすめします。

見つかったアカウント数

⚠ あくまで見つかった数であり、存在する全てのスパムアカウントを網羅したわけではありません。

スパムからいいねが来たら…

  • スパムアカウントの ID を #いいねスパム タグ でシェアしてください。情報提供にご協力お願いします。
    • スクリーンショットではなく ID をテキスト形式で貼ってもらえると助かります。
  • 匿名で報告したい場合は こちらの Google Form をご利用ください。
  • 凍結されているのを見つけた場合もお知らせ頂けると助かります。

アカウント一覧 (参考)

これまでに観測されたスパムアカウントを 20 件ずつのグループにまとめて掲載しています。 PC 向けの 表形式版 もあります。

⚠ ブロックや X への報告が目的であれば、まずは直近で活動しているアカウントを #いいねスパム タグ を使って探してください。 アカウントの総数は膨大で、全てを一気にブロックしたり報告しようとするのは無謀です。 X では短時間に数十以上のアカウントをブロックすると閲覧が制限されたり強制ログアウトさせられたりします。 もし連続でブロックしたり報告したりする場合は、20～40 件毎にしばらく時間を空けてください。

スマホアプリの場合は各リンクをクリック後「ユーザー」のタブを選択してください。
