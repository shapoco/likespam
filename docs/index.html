いいねスパム一覧

概要

X において不特定多数に対して「いいね」通知を送りつけて 当選詐欺 (?) に誘導しようとするスパムアカウントをまとめたものです。 管理者は勝手に「いいねスパム」と呼んでいます。以前はアカウント名に【公式】と付けていたことから、X では「公式スパム」などとも呼ばれています。

使用されているアイコン:

このページでは X で観測されたスパムアカウントを収集しています。

これらのアカウントは1万個以上あり、もはやひとつひとつブロックすることに意味があるのかは微妙です。 報告するのであれば、詐欺の窓口となっている @Zy_Iove とその周辺のアカウント を報告することをおすすめします。

見つかったアカウント数

あくまで見つかった数であり、存在する全てのスパムアカウントを網羅したわけではありません。

スパムからいいねが来たら…

  • スパムアカウントの ID を #いいねスパム タグ でシェアしてください。
    • スクリーンショットではなく ID をテキスト形式で貼ってもらえると助かります。
  • 匿名で報告したい場合は こちらの Google Form をご利用ください。
  • 凍結されているのを見つけた場合もお知らせ頂けると助かります。

アカウント一覧

スマホアプリの場合 1 アカウントずつリンクを開くのは面倒なので 20 アカウントずつの検索リンクにしています。 PC 向けの 表形式版 もあります。

アカウントは膨大で、全てを一気にブロックしたり報告しようとするのは無謀です。 X では短時間に数十以上のアカウントをブロックすると閲覧が制限されたり強制ログアウトさせられたりします。 連続でブロックや報告を行う場合は 20～40 件毎にしばらく時間を空けてください。

ブロックや X への報告が目的であれば、まずは直近で活動しているアカウントを #いいねスパム タグ を使って探してください。 「最新」タブに結果が表示されない場合は、検索キーワードに「since:2024-9-25」などを付け加えて日付を指定し 「話題のツイート」タブに切り替えると最近のポストを閲覧できます (ただし時系列順にはなりません)。

スマホアプリの場合は各リンクをクリック後「ユーザー」のタブを選択してください。

X 検索の仕様のためスパムではない無関係なアカウントが混ざることがあります。
