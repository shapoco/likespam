いいねスパム一覧

概要

X において不特定多数に対して「いいね」通知を送りつけて詐欺 (?) に誘導しようとするスパムアカウントをまとめたものです。 管理者は勝手に「いいねスパム」と呼んでいます。X では「公式スパム」などとも呼ばれています。 先行ブロックや報告にご活用ください。

いいねスパムの見つけ方

  • X検索: "個された人"
  • X検索: "特別なご案内です"
  • Emilydanie29864氏作成のリスト

スパムからいいねが来たら…

  • #いいねスパムタグ でアカウントの ID をポストしてくれれば拾います。
  • 凍結されているのを見つけた場合もお知らせ頂けると助かります。
  • 匿名で報告したい場合は こちらの Google Form をご利用ください。

使い方

スパムアカウントを20件ずつのグループにまとめています。

  1. リンクをクリックすると X の検索結果が表示されます。
  2. スマホアプリの場合は「ユーザー」のタブを選択してください。
  3. 表示されたアカウントをひとつずつブロックまたは報告してください。
  • スパムとして報告するよりも「攻撃的な行為や嫌がらせ」→「特定ユーザーへの嫌がらせ」の方が効果的という説があります。
  • 短時間に大量（50 件くらい？）のアカウントをブロックすると、一時的に X の閲覧を制限されたり強制的にログアウトされる場合があります。 不審なアクティビティとみなされる可能性があるため、一度に 40 件程度に留めた方が良さそうです。
